About School Performance Tracker

School Performance Tracker waa nidaam fudud oo casri ah oo lagu xisaabiyo laguna la socdo waxqabadka ardayda. Nidaamkani wuxuu ka caawiyaa macallimiinta iyo maamulka iskuulka inay si degdeg ah u arkaan celceliska dhibcaha iyo darajada arday kasta.

Mashruucan waxaa lagu dhisay HTML, CSS, iyo JavaScript si loo muujiyo adeegsiga DOM manipulation, event handling, input validation, iyo kaydinta xogta iyadoo la adeegsanayo arrays. Natiijooyinka waxaa lagu soo bandhigaa isla bogga iyada oo aan dib loo cusboonaysiin (reload) page-ka.

Ujeeddada ugu weyn waa in la sameeyo hab sahlan oo lagu qiimeeyo ardayda laguna yareeyo qaladaadka xisaabinta gacanta lagu sameeyo.
